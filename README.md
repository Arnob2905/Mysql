#  Data Transformer – SQL Project

This project demonstrates SQL operations such as joins, subqueries, date functions, string manipulation, window functions, and conditional logic.  
It uses a sample database named **datatransformer** with three main tables:

- **Customers**
- **Orders**
- **Employees**

---

##  1. Database Setup

The project starts by creating the database and inserting sample records.

### **Tables Included**
- `Customers` – Stores customer details (name, email, registration date)
- `Orders` – Stores customer orders and amounts
- `Employees` – Stores employee info such as department, hire date, and salary
